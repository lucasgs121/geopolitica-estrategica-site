# Como rodar com o painel Mercados Estratégicos funcionando

1) Abra esta pasta no VS Code
2) No terminal, dentro da pasta `TESTE`, rode:

npm install
npm start

3) Abra no navegador:
http://localhost:3000

Importante: se você abrir o `index.html` direto (file://) ou usar Live Server sem rodar o `npm start`,
o endpoint `/api/markets` não existe e os itens podem aparecer como "Indisponível".
